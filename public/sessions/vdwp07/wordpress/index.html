WordPress

  • Started life as a blogging tool in 2003
  • Developed into a feature-rich CMS
    • Current version: 4.9.4 (February 2018)
  • Used by over 29% of "top 10 million" websites and 60 million sites in total (source)
  • Over 54,000+ plugins
  • Thousands of themes
  • WordPress.com - hosted blogging service
  • WordPress.org - blogging software

WordPress requirements

  • PHP 7.2 or greater
  • MySQL 5.6 or greater OR MariaDB 10.0 or greater
  • Nginx or Apache with mod_rewrite module

Creating a WordPress theme

Using Underscores (_s) theme from Automattic as a base.

  1. Download _s theme
  2. Unzip files
  3. Install theme on your WordPress site
    • FTP to: {blog_directory}/wp-content/themes/
    • Go to 'Appearance > Themes' in WordPress admin area
    • Activate your theme
