WDUX01 Reading

After the WDUX01 session, you may like to look at some of these resources.

If you're not very confident with HTML then you must read these tutorials.

  • Introduction to HTML
  • Getting started with HTML
  • HTML School

There's really no need to spend money buying books about learning HTML because all the information you need is available online. But if you work offline a lot and need a good HTML reference book then I recommend:

  • HTML and CSS: Visual QuickStart Guide

If you desperately want a book about learning HTML then I recommend:

  • Sams Teach Yourself HTML and CSS in 24 Hours

The definitive reference for what's allowed in HTML 4 (the version of HTML we've looked at so far) is:

  • HTML 4.01 Specification

It's not what you'd call light reading though.
