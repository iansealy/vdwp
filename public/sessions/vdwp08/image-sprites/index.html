Image Sprites

An image sprite is a collection of images put into a single image. Image sprites reduce the number of server requests and are often used for small images used in the user interface.

Example:

  1. Wiggle
  2. Image sprite 1
  3. Image sprite 2

Using Image Sprites

The image sprite is put into the background of an element using the background-image: url() property and then the appropriate part of the image is displayed using a combination of:

  • background-postion
  • background-repeat
