Assignment

The Visual Design and Web Project assignment must be submitted by Monday, Apr 23 2018.

The assignment is split into two equally weighted parts - Visual Design and Web Project.

Visual Design

You will produce a portfolio of images demonstrating how to manipulate media for use on the Web. The deliverables are:

  1. A portfolio of 10 digital images, ideally for use in your final Web Project. The images will be taken from a variety of sources including scanned images, original artwork created in a graphics package and images from an online photo bureau. For each image, briefly explain, in a caption, why and how you produced it. You will show that you understand the following principles:
    • colour correction and manipulation
    • saving images in a Web format
    • optimising graphics for Web use
  2. Two user interface mockups as prototypes for your Web Project. The mockups will include text, colour and images.
  3. Either
    • An image sprite and a sample page showing how it will be used.
    or
    • An animated GIF for use in a social media campaign. You will describe how the image will be used as part of the campaign.

Web Project

You will produce an accessible, professional Web site that conforms to Web standards and best practice. The Web site will be accompanied by supporting documentation describing how and why you've developed your site in the way you have.

Your Web site will:

  • Conform to Web standards and best practice
  • Have an appropriate information architecture
  • Use appropriate user interface elements in a consistent design
  • Demonstrate good separation of content (HTML), presentation (CSS) and, if appropriate, behaviour (JavaScript)
  • Demonstrate a sound understanding of HTML and CSS
  • Be demonstrably accessible
  • Be appropriate to the subject matter and audience
  • Be well documented with comments in the code and detailed supporting documentation

(Note: marks will be available for the use of advanced client-side (e.g. JavaScript) and server-side (e.g. SSIs) techniques, but the majority of the marks are focussed on the above and the use of such advanced techniques is not a requirement for a good overall mark.)

The supporting documentation is very important for this assignment as it allows you to demonstrate wider reading and development that did not make it into the final Web site. (For example, you might have developed an advanced mega menu for your site, but found that it was impossible to implement in an accessible way. Although not suitable for your final Web site, the code could still be included in your documentation and discussed, as this is likely to demonstrate your understanding of HTML, CSS, JavaScript and accessibility.)

Submission

Your final submission for this assignment will take the form of a Word or PDF document containing:

  • The URL(s) in your Web space of the Visual Design deliverables
    (e.g. http://iansealy.vdwp1718.bbkweb.org/portfolio/)
  • The URL of your Web Project
    (e.g. http://iansealy.vdwp1718.bbkweb.org/webproject/)
  • The supporting documentation for your Web Project
