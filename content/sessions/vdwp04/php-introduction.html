What language is best?

  • It doesn't matter... sort of
  • All have pros and cons and most programmers will tell you their language of choice is the best
  • Considerations for Web development:
    • Server operating system (e.g. Linux/UNIX or Microsoft)
    • Web server software (e.g. Apache, IIS, Nginx etc)
    • Database software (MySQL, PostgreSQL)
    • Cost (hosting and licences)
    • Ease of setup
    • Preference
  • Once you've learned to program in one language, learning a new language isn't hard
  • Pick the best solution for a problem, don't feel like you have to use the same tool each time

What is PHP?

  • PHP - PHP: Hypertext Preprocessor (was: Personal Home Page)
  • It's really popular:
    • Used by 83% of all web sites
  • Used for popular tools: WordPress, Drupal, Moodle, Joomla
  • Used (amongst others) by: Facebook, Yahoo, Wikipedia, Wordpress.com, Flickr
  • Easy to set up
  • Integrated database support
  • Not without its detractors

Server-side processing

  • Recap of what happens on the server
    1. Request sent to Web server
    2. In the case of PHP, Web server runs PHP scripts using a PHP interpreter
    3. Interpreter returns output from PHP script to Web server
    4. Web server sends output to browser
  • Unlike CGI scripts, you can mix HTML and PHP code in the same file... although most developers don't and prefer to keep them separate

Syntax basics

  • Script structure: 
        <?php
    
    ?>
  • Most lines must end with a semi-colon ;
  • Variables - containers for storing data
    • Start with a $ and then a letter or underscore (not a number)
    • Variable names can only contain A-Za-z0-9_
    • Case sensitive

Syntax basics

  • Example variables:
    • Integer
      $n = 5;
    • Float
      $pi = 3.1415;
    • String
      $my_name = "Kieren";
    • Boolean
      $has_paid = true;
    • Don't have to declare the data type
  • Comments: //, # or /* and */
  • Echo - output some information:
    echo "<p>Hi there!</p>";

Operators

  • +, -, /, *
  • assignment =
  • equal ==
  • identical ===
    • Use === if you want to check if two things are identical. If you use == then, for example, 0 and false are considered equal.
  • not equal !=
  • not identical !==
  • increment += or ++
  • decrement -= or --

Arrays

  • Arrays are lists of 'things'
  • Order of items is preserved
  • Creating an array:
    $colours = array("red","orange","green","blue");
  • Accessing elements (zero-indexed): $colours[0]; // red
  • Number of items in array: count($colours);
  • Add items: array_push($colours, "white", "black");
  • Delete 1 item at index position 2 in the array: array_splice($colours, 2, 1);
  • For debugging you can print the whole array: print_r($colours);
  • Many array functions

Associative array/hash/dictionary

  • In PHP there's just one array type and you can use it as an integer indexed array or use other data types (e.g. strings) for the indices
  • In other languages arrays/lists are different from dictionaries/associative arrays
  • $price = array(
	"Cake"=>"2",
	"Jam"=>"1.50",
	"Biscuits"=>"0.5"
);
  • Accessing elements
    $price['Cake']; // 2
  • Adding elements: $price['Sweets'] = "0.2";
  • Deleting elements: unset($price['Sweets']);

For loops

  • Counting to 5 from 0 
    for ($x=0; $x<=5; $x++) {
	echo "The number is: $x<br>";
}
  • Looping over array 
    $colours = array("red","orange","green","blue");
foreach ($colours as $colour_name) {
	echo $colour_name;
}
  • Looping over associative array 
    $links = array(
	"Google"=>"http://www.google.com",
	"Yahoo!"=>"http://www.yahoo.com",
	"Bing"=>"http://www.bing.com"
);
foreach ($links as $key => $value){
    echo "<a href=\"$value\">$key</a>";
}

Form data

  • Form in the HTML:
    <form action="example.php" method="GET">
    <input type="text" name="your_name">
    <input type="submit" value="Submit">
    </form>
  • Access the data in your 'example.php' PHP script
    $_GET["your_name"]
  • An example of using the form data: 
    <?php
if ($_GET["your_name"] == 'Bob') {
    echo "Hi Bob, welcome back";
} else {
    echo "Hello new visitor";
}
?>

Form data

  • If using POST then:
    $_POST["your_name"]
  • Sanitise any data from users, e.g. remove anything that's not a character within these ranges A-Z, a-z or 0-9:
    $sanitised_string = preg_replace("/[^A-Za-z0-9]/", '', $string);
