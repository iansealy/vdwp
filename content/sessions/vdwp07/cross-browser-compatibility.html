Introduction

  • Best method to avoid browser incompatibilities is to develop using Web standards
  • Even then can still get differences
  • Recommend developing using a highly standards-compliant browser (personally Chrome or Firefox) and then fix problems in IE
  • [Focusing on CSS differences (style); for JavaScript differences (behaviour) advise using a JavaScript library, like jQuery]

Tools

Tools to aid Web development:

  • Web Developer (Firefox extension and now Chrome and Opera)
  • Firefox, Chrome & Edge DevTools
  • Opera Dragonfly
  • Safari Web Inspector
  • IE8+ Tools

More Tools

Testing in other browsers:

  • IE NetRenderer (free)
  • Browsershots (free)
  • BrowserStack (not free)
  • IE & Edge Virtual Machines (free)
  • Litmus (not free; email testing)

Valid HTML And CSS

  • Invalid HTML or CSS often causes inconsistencies
  • Solution: Validate HTML and CSS before investigating other causes

Reset CSS

  • Different browsers have different default CSS
  • Solution: Use Reset CSS (e.g. Eric Meyer's) or normalize.css

Quirks Mode

  • Quirks mode emulates behaviour of old browsers for backwards compatibility
  • Many differences between quirks mode and standards mode
  • Most commonly noticed difference is due to different box models
  • Example
  • Solution: Always use a DOCTYPE that forces standards mode
  • Fixed example

Almost Standards Mode

  • Most DOCTYPEs actually force almost standards mode rather than standards mode
  • In standards mode, default for images is display: inline;, so small space appears below
  • Example
  • Solution: display: block;
  • Fixed example

Modernizr

  • Gradual implementation of HTML5 and CSS3 by browsers means not all features implemented in all browsers
  • Solution: Use Modernizr
  • Example
  • Tests for features and creates a JavaScript object and adds classes to html tag indicating which features are supported
  • Can write different JavaScript and CSS depending on whether features are available
  • JavaScript polyfills implement features in browsers that don't natively support them
