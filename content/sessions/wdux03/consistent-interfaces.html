Importance

  • A consistent user interface is one where all the navigational tools are in a similar place on every page of a site and where all the pages have a similar look and feel
  • A consistent user interface allows users to complete their task more quickly and easily
  • An inconsistent or deficient user interface confuses users ("Am I still on the same site?"), looks unprofessional and may force them to leave
    • University of Bath
  • Main difference between professional and amateur sites
  • A user interface must be planned and requires knowledge of all the content and possible content of the site

Creating a nav bar in CSS

  • Nav bars (and footers) are essentially unordered lists of links
  • Default rendering is for list items to stack vertically. For example:
  <ul id="nav">
  <li><a href="#">About</a></li>
  <li><a href="#">News</a></li>
  <li><a href="#">Sport</a></li>
  <li><a href="#">Shop</a></li>
  </ul>

Creating a nav bar in CSS - 2

  • Use: display: inline to get the list items on one line:
    ul#nav li {
      display:inline;
    }

Creating a nav bar in CSS - 3

  • Now we can add some colour and padding:
  ul#nav li {
    display:inline;
    list-style-type: none;
    margin : 0;
    padding:0;
  }

  ul#nav li a {
    float: left;
    background-color : #BDDEFF;
    text-decoration: none;
    color: #000000;
    margin:0;
    padding : 6px 20px 6px 20px;
  }

  ul#nav li a:hover {
    background-color : #566B80;
    text-decoration: underline;
    color: #ffffff;
  }
