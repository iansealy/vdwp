Have a go at animating the images in this example page. You can download all of the resources in a zip file.

The background images are 240px x 120px.

You'll need the following CSS properties:

  • animation - to name the animation
  • animation-duration - to set how long the animation runs for in seconds (2s)
  • animation-iteration-count - to set how many times the animation runs
  • animation-timing-function - to set whether the animation runs evenly or eases in or out
