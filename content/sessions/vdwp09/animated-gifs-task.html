Have a go at creating an animated GIF using Photoshop that displays a text message in sequence.

  1. Create a new document in Photoshop using one of the standard banner sizes
  2. Add the text for your message putting each part of the message on a different text layer
  3. Use the Frame Animation Window to add frames to your animation
  4. Use show and hide layers to animate the appearance of the text
