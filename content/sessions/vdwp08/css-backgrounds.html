CSS Backgrounds

CSS provides a number of background-related properties. These include:

  • background-color
  • background
  • background-image

background-color

  • body { background-color: red;}
    Fills the background of an element with a named color. Example
  • body { background-color: #ff0000; }
    Fills the background of an element with a colour specified using a hex value. Example
  • body { background-color: rgb(201, 76, 76); }
    Fills the background of an element with a colour specified using an RGB value. Example
  • body { background-color: rgba(201, 76, 76, 0.5); }
    Fills the background of an element with a colour specified using an RGB value and an alpha (transparency) value. Example

Background And Gradients

The background shorthand property makes it possible to set all background properties in one declaration. It also allows you to create a gradient in the background of an element.

  • body {background: linear-gradient(red, yellow);}
    Creates a top to bottom gradient from red to yellow. Example
  • body { background: linear-gradient(to right, red , yellow);}
    Creates a left to right gradient from red to yellow. Example
