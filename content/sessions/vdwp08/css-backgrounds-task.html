Using Firefox or Chrome and the Web Developer toolbar:

  • Open a random sample of Web pages.
  • Use the 'Hide Background Images' option in the Web Developer toolbar to hide and show background images on a page.
  • Use 'Disable All Styles' to turn off the CSS to see whether the page still makes sense.
  • You could also have a go at saving some of the background images you find to use in the next task. In Firefox you can right click on a background image and choose the option 'View Background Image' then Ctrl-S to save it
